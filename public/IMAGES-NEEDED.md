# Required Images for Salty Peak Solutions

Please add the following images to the `/public` directory:

## 1. Logo Image
- **Filename:** `logo.png`
- **Description:** Salty Peaks Solutions logo with orange mountain icon
- **Recommended Size:** ~300px width (will be displayed at 150px)
- **Format:** PNG with transparent background preferred

## 2. Hero Image
- **Filename:** `salty-peaks-solutions-hero.png`
- **Description:** Fire truck with MagneGrip exhaust system
- **Recommended Size:** 1000px x 1000px minimum (square aspect ratio)
- **Format:** PNG or JPG
- **Notes:** This image will be displayed in a rounded container with an orange border

## Placeholder Images
Until the actual images are added, you can:
1. Use placeholder image services like https://via.placeholder.com/
2. Or temporarily comment out the `<img>` tags in the components

## Location
All images should be placed in: `/public/` directory
- `/public/logo.png`
- `/public/salty-peaks-solutions-hero.png`
